Ellis has been chatting to Blake about the latest project their company is starting. The discussion reaches a crucial point.

"How long will the project take?" Ellis asks.

"Well," Blake replies. "The first 90% of the code will account for the first 90% of the time..."

He pauses for a beat, then continues.

"And then the project will get canned!"

:::

> The first 90 percent of the code accounts for the first 90 percent of the development time. The remaining 10 percent of the code accounts for the other 90 percent of the development time.
> <cite>Tom Cargill</cite>
